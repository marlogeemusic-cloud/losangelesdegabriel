# Gabriel's Angels Foundation · Fundación Los Ángeles de Gabriel

Bilingual (EN/ES) landing site for **Gabriel's Angels Foundation** — a non-profit in Houston, TX, founded in December 2014 in memory of **Gabriel Ángel Maciel**. The foundation gives dignified, immediate help with funeral, medical and daily expenses for low-income families — *no bureaucracy, no hard questions*.

> **Together We Can · Juntos Podemos**

## Features
- 🌐 Bilingual: English by default, one-tap toggle to Spanish (remembers your choice).
- 📖 Multi-page experience (single-file SPA): Home · Marlogee & Gabriel · How We Help · Donate · Get Help.
- 💝 Donations: Cash App (`$LosAngelesDeGabriel` + QR) and Zelle (`(713) 494-2268`) with tap-to-copy.
- 🤝 Transparency model: the foundation organizes the benefit, the family collects directly.
- ✨ Celestial design with soft animations (fade-in, drifting clouds, gentle float).
- 📱 Fully responsive, mobile-first.

## Stack
Plain HTML + Tailwind CSS (CDN) + vanilla JS. No build step — deploys as a static site.

## Run locally
Just open `index.html` in a browser, or serve the folder:
```bash
npx serve .
```

## Deploy
Static site — deploys to Vercel with zero configuration (`vercel.json` included for clean URLs and image caching).
